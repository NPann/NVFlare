class InTimeAccumulateWeightedAggregator(exclude_vars=None, aggregation_weights=None)¶

Bases: nvflare.apis.aggregator.Aggregator

Perform accumulated weighted aggregation It computes weighted_sum = sum(shareable*n_iteration*aggregation_weights) and sum_of_weights = sum(n_iteration)

in accept function The aggregate function returns weighted_sum / sum_of_weights

Parameters

  • exclude_vars ([type], optional) – regex to match excluded vars during aggregation. Defaults to None.

  • aggregation_weights ([type], optional) – dictionary to map client name to its aggregation weights. Defaults to None.
