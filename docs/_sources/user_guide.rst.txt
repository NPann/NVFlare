##########
User Guide
##########

This user guide shows how to use NVFlare to deploy and operate an FL system on multiple sites with a Provision-Start-Operating procedure.

You can also check some example applications integration like `Clara Train <https://docs.nvidia.com/clara/clara-train-sdk/>`_
and `MONAI <https://github.com/Project-MONAI/tutorials/tree/master/federated_learning/nvflare>`_
to see the capabilities of the system and how it can be operated.
